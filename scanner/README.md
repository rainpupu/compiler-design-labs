# 实验二选做：Vue + Vite Scanner 可视化界面

## 功能

- VSCode 风格页面
- 黑色 / 白色主题切换
- 输入代码高亮
- Token 流输出
- Token 表格
- 下载 `token.txt`
- 点击 Token 展示 DFA 路径
- 自动生成 DFA 转移图
- 生成 DOT 描述

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开终端中显示的地址，通常是：

```text
http://localhost:5173
```

## 构建发布版本

```bash
npm run build
npm run preview
```
